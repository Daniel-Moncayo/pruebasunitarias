# Smart Wallet - Pruebas Unitarias

**Nombre:** Daniel Moncayo

Proyecto en Java 17 con Maven y JUnit 5 para validar reglas de negocio de la clase `SmartWallet`.

## Requisitos
- Java 17+
- Maven 3.8+

## Ejecutar pruebas
```bash
mvn test
```

## Estructura principal
- `src/main/java/com/example/wallet/SmartWallet.java`
- `src/test/java/com/example/wallet/SmartWalletTest.java`
- `pom.xml`
